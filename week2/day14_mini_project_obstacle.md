# Day 14 — 미니 프로젝트: LIDAR 장애물 회피

> **RPi 버전 Day 14와 내용이 동일합니다.**
>
> Jetson Nano에서 장애물 회피 노드는 Docker 컨테이너 내부에서 개발/실행합니다.
> 시각화는 Remote PC의 RViz2에서 확인합니다.

---

## 실행 구성

```
┌─────────────────┐  /scan, /odom     ┌──────────────────┐
│  Docker 컨테이너 │ ◄─────────────── │  Jetson Nano     │
│  (ROS2 Humble)  │                   │  (네이티브)       │
│                 │  /cmd_vel         │                  │
│  장애물 회피 노드│ ────────────────► │  Bringup → 모터  │
└─────────────────┘                   └──────────────────┘
         │                                   │
         │ /avoid_status                     │ /scan, /odom
         ▼                                   ▼
    ┌──────────────────────────────────────────┐
    │           Remote PC (RViz2)               │
    │  - LaserScan 표시                        │
    │  - 장애물 회피 상태 표시                  │
    └──────────────────────────────────────────┘
```
